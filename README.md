Our project was Instagram.The core features for our project were :
1.Feed Generation:Displays posts from followers first, followed by recommended posts based on a simple algorithm considering engagement, relevance, and user interests.
2.Posting:Allows users to upload content (images, text, etc.), add captions, and edit their own posts.
3.Feed Interaction:Enables users to like, comment on, and repost  other users' content.
4.Friend Suggestion:Recommends potential friends/followers based on mutual connections
We implemented bpmn,usecase,class,mockUI,collaboration,state,sequence diagram and posting feature for Instagram.
